字面量

  1. 定义：
    • 被写下来的固定的值，称为字面量
  2. 常用值类型：

运算符

  1. 算术运算符：
    • 加：+
    • 减：-

  2. 比较运算符：

    • 小于：<

    • 小于等于：<=

字符串

  1. 定义：
    • 单引号：name = ‘K’
    • 双引号：name = “K”
    • 三引号：name = “””K”””
  2. 拼接：
    • 字符串之间可以通过 “+”来拼接
    • 字符串也可格式化拼接
  3. 精度控制：
    • “m.n”控制数据宽度和精度，m控制宽度(小于本身时不生效)，n控制精度(四舍五入)。

      %5.2f：宽度为5，小数点精度为：2
      %5的：只设置宽度为5

  1. 定义：
  2. 特点：
    • 容纳任意数量字符串
    • 有序存储，拥有下标
    • 无法修改
    • 允许字符重复
  3. 访问：
  4. 使用：
    • 查询某字符下标
      目标串.index(‘目标字符串’)，返回起始字符下标值或者报错ValueError
    • 替换待修改串为新字符串
      新串 = 目标串.replace(‘待修改串’，’新字符串’)
    • 将字符串按’某串’分割为多个字符串，并返回至列表对象
      新列表 = 目标串.split(‘某串’)
    • 去除字符串前后所有’指定元素’,得到一个新字符串
      新串 = 目标串.strip(‘指定元素’)
    • 统计指定字符在目标串中的数量
      目标串.count(‘指定字符串’)
    • 统计串中总的字符数量
      len(‘目标串’)
